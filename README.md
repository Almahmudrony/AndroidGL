##OpenGLJava

##OpenGLNDK
[VSL](http://www.lighthouse3d.com/very-simple-libs/) has been ported to Android. This is a C++ demo app, using NDK, for OpenGL 3.0. It includes Assimp for model loading. Texture loading is achieved with JNI.
